# 🕵️ CyberHound OSINT Bot

Powerful Discord bot with people search, breach checks, and more!

## Commands
- `!find <name>` - Search for a person
- `!help` - Show help

## Setup
1. Add your API keys to Render environment variables
2. Deploy!

## Environment Variables Required
- `DISCORD_TOKEN`
- `APOLLO_API_KEY`
- `GOOGLE_API_KEY` (optional)
